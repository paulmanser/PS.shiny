Input variables on the Regression 1 dialog box

α
The Type I error probability for a two sided test. This is the probability that we will falsely reject the null hypothesis.

n
The number of patients who must be studied to detect a true regression slope λ with the specified power and Type I error probability α.

power
The probability of correctly rejecting the null hypothesis that the regression line has slope λ0=0 given the specified sample size, Type I error probability α, and a true regression slope of λ.

λ
The slope of the linear regression line that we wish to detect.

σ
The standard deviation of the regression errors. The regression error for a given patient is the difference between her observed response and her expected response given the value of her independent variable.

σx
The standard deviation of the independent variable.
