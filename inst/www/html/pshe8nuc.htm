Input variables on the Regression 2 dialog box

α
The Type I error probability for a two sided test. This is the probability that we will falsely reject the null hypothesis.

n
Number of experimental (group 2) subjects who must be studied.

power
Probability of correctly rejecting the null hypothesis that the regression lines have equal slopes (or equal y intercepts) given n experimental patients studied, m controls per experimental subject, Type I error probability α, and a true difference in the regression slopes λ2-λ1 (or y intercepts γ2-γ1).

m
Number of controls per experimental subject (ratio of number of group 1 to group 2 subjects).

σ
The standard deviation of the regression errors. The regression error for a given patient is the difference between her observed response and her expected response given the value of her independent variable.

σx1
The standard deviation of the independent variable in the control group (group 1).

σx2
The standard deviation of the independent variable in the experimental group (group 2).

λ2 - λ1
Difference in the slopes of the regression lines for the experimental and control groups that we wish to detect.

γ2 - γ1
The difference in the y intercepts of the regression lines for the experimental and control groups that we wish to detect.

x1
An estimate of the average value of the independent variable for control (group 1) subjects.

x2
An estimate of the average value of the independent variable for experimental (group 2) subjects.
