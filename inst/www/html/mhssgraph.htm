Sample Size Graphs for Mantel-Haenszel Tests

This window is used to produce plots of sample size vs. alternative hypotheses or sample size vs. power for Mantel-Haenszel Tests. The alternative hypotheses may be expressed as an (adjusted) odds ratio. The variable that is not chosen for the x-axis is held constant for each sample size curve. You may plot multiple curves on the same graph with different values of this third variable.
