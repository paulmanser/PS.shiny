Linear Regression Overview (2 Treatments)

Sample size and power calculations for studies involving linear regression. This module is for studies involving two independent linear regressions and you wish to compare the slopes or intercepts of these regression lines. The values of the independent (x) variables of the regression lines may either be specified by the investigator or determined observationally when the study is performed. In the latter case, the investigator must estimate the standard deviation of the independent variables.

The Regression 2 dialog box has four sections: Output, Design, Input, and Description.

Each null hypothesis is tested with respect to a two-sided alternative hypothesis.
