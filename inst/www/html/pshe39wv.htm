Sample Size Graphs for Linear Regression (2 Treatment)

This window is used to produce plots of sample size vs. alternative hypotheses or sample size vs. power for survival studies. The alternative hypotheses may be expressed as the difference in slops of two independent simple linear regression curves, or the difference in y-intercepts of these curves. The variable that is not chosen for the x axis is held constant for each sample size curve. You may plot multiple curves on the same graph with different values of this third variable.
