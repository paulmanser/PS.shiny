Output section of the Dichotomous dialog box

The Output section is where the user selects the result to be generated by the PS program. In the Output field select one of the following (to make a selection, click in the Output box then click the desired output).

Sample size

The sample size needed to detect a specific alternative hypothesis with a specified power and Type I error probability α.

Power

The power with which a given sample size can detect a specific alternative hypothesis with a specific Type I error probability α.

Detectable alternative

The alternative hypothesis that can be detected with a specified level of power given a specified sample size and Type I error probability α.
