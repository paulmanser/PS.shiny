Plotting

To plot a graph you must

  1. Select a tab on the main PS window to specify the study design.
  2. Specify the desired output in the "What do you want to know?" field. This choice determines one of the axes of the graphic.
  3. Complete the data entry for your study design and press the calculate button. The Graphs button will darken to indicate that you may now proceed to the graphics window.
  4. Press the Graphs button. The graphics dialog box will appear.
  5. If you selected Sample size or Power in step 2, you will be asked to specify which variable you want on the x axis. Make this choice and provide the range of values for this axis on your graph. The y axis will be determined by your choice in step 2. Specify a range for the y axis. The Plot button will darken to indicate that all the needed data has been supplied. Another field will appear specifying the value of the variable that was not selected for the x axis. The default value of this variable is what it was after step 3. You may alter this value if you wish.

Press the Plot button. The desired graphic will appear. In this graph, the variable that was not specified for the x axis will be held constant at the indicated value. A figure legend will give this value. Other parameter values that were specified in the main window will be written near the bottom of the graphic. Click on parameter definitions near the bottom of the graphics window to see the definitions of these parameters.

Multiple Lines on One Graph

We can show the interactions between the power, sample size and detectable alternative hypothesis of a study by plotting multiple lines on one graph. For example, we might plot sample size against detectable alternative hypothesis holding power constant; our graph might have three such curves with power set to 0.80, 0.90, and 0.95, respectively. To do this with the PS program you plot one graph, change the value of the variable that is not on either axis and press the Plot button. A new line and figure legend will be added to you graphic. You may repeat this process as often as desired.

Logarithmic Axes

You can plot either X, Y or both axes on a logarithmic scale. Let us refer to the axis that you wish to draw on a logarithmic scale the logarithmic axis. To create your graph you must

  1. Plot the desired graph on the default (linear) scale. For best results choose the range of the logarithmic axis to be 1% less than the desired range in your final graphic.
  2. Double click on the logarithmic axis. The Chart Designer dialog box will appear. The logarithmic axis should be highlighted in the large white field on the left of this dialog box.
  3. Click on the Scale Type tab.
  4. Click the logarithmic selection. Change the log base if you do not want a log base 10 scale.
  5. Press the Apply button. Then press OK. The desired graphic should appear. If the range of the logarithmic axis is too large, repeat the above procedure using a slightly smaller range for this variable.

You may need to do some minor editing of your graphic in a graphics editor. For example, suppose that the logarithmic axis is the Y axis and denotes sample size, you are plotting sample size against detectable difference for t-tests, and the ranges of the X and Y axes are from 0 to 5 and from 0 to 1000, respectively. The resulting sample size curve will be correctly plotted for all values of X that correspond to sample sizes less than 1000. For X between 0 and the true difference that can be detected with sample size 1000, there will be a spurious horizontal line at Y = 1000. This spurious horizontal line should be deleted with a graphics editor before using this graph in a presentation of publication. This spurious line only occurs when one of the axes is plotted on a logarithmic scale.

Editing

PS graphs are generated with the First Impression graphics control. This software permits the user considerable freedom in customizing her graphics. For detailed help on this software create a graph following the directions given above and then double click on different components of the graphic. Various context specific dialog boxes will appear. Highlighted text in the white box on the left of the dialog box indicates which component of the graph you have selected. A help button at the lower right of these boxes will provide detailed guidance on using this software. What follows is far from complete but outlines some of the more useful modifications that you can make to PS graphics.

  • Modifying a power or sample size curve: Double click on the curve. The Chart Designer dialog box for the selected curve (series) will appear. Selecting the Lines tab will permit you to change the color, thickness or style of the selected line.
  • Deleting a curve: Double click on the curve and check the hide series box on the Options tab of the Chart Designer dialog box.
  • Modifying the axis titles: Double click on the axis title. Select the text tab to modify the title. Select the font tab to modify the font, style, color or size of the title.
  • Modifying the axis labels: Double click on an axis label. Select the font tab to modify the font, style, color or size of the label.
  • Deleting the figure legend or footnote: Double click on the text to be deleted. On the location tab click in the Visible check box controls whether the selected legend or footnote is displayed on the graph.
  • Using your own graphics editor: It is also likely that you can use your favorite graphics editor to modify your graphic. Right-click on the graphic and select copy. Then paste the graph into your editor. With Microsoft PowerPoint 97 you must then ungroup the pasted graph before editing it. In some editors you may need to resize the graph before ungrouping it.

Saving

Pressing the Save button on the graphics window pops up a dialog box that allows you to specify the file name and folder where your graphic is to be saved. PS graphs are saved in Windows Metafile format. Saving and then opening a PS graph is an alternate route for exporting PS graphs to your graphic editor.

Printing

Pressing the Print button on the graphics window pops up a standard windows print dialog box.
