Output section of the Regression 2 dialog box

The Output section is where the user selects the result to be generated by the PS program. In the Output field select one of the following (to make a selection, click in the Output box then click the desired output).

Sample size

Number of experimental patients n who must be recruited to detect a true difference in regression slopes λ2-λ1 with the specified power given m controls per experimental subject, and Type I error probability α. λ1 and λ2 are the regression slopes of the control and experimental groups, respectively.

Power

Probability of correctly rejecting the null hypothesis that the regression lines have equal slopes given n experimental patients, m control patients per experimental patient, Type I error probability α, and a true difference in regression slopes λ2-λ1.

Detectable alternative

The true difference in regression slopes that can be detected with the specified power given n experimental patients, m controls per experimental patient, and Type I error probability α.

Analogous sample size, power, and detectable difference calculations can be done in terms of the intercept difference γ2-γ1.
