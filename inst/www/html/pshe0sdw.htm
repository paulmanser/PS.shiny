Output section of the Regression 1 dialog box

The Output section is where the user selects the result to be generated by the PS program. In the Output field select one of the following (to make a selection, click in the Output box then click the desired output).

Sample size

The number of patients who must be studied to detect a true regression slope λ with the specified power and Type I error probability α.

Power

The probability of correctly rejecting the null hypothesis that the regression line has slope λ0=0 given the specified sample size, Type I error probability α, and a true regression slope of λ.

Detectable difference

The slope of the regression line that can be detected with the specified power and sample size given a Type I error probability α.
